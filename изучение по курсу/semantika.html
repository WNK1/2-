CSS Flexbox — это технология для создания сложных гибких макетов за счёт правильного размещения элементов
на странице. О самой технологии пишут здесь. Мы же решили объяснить возможности CSS Flexbox
использованием гифок — настоящая flex шпаргалка.

  1. display:flex
  2. flex-direction
  3. justify-content
  4. align-items
  5. align-self
  6. flex-basis
